📄 Formato del archivo JSON para tus propios ejercicios 
[
  {
    "id": 1,
    "audioFile": "audio1.mp3",
    "audioText": "nǐ hǎo",
    "question": "¿Qué significa este saludo?",
    "correct": 0,
    "options": [
      { "zh": "你好", "py": "nǐ hǎo", "es": "Hola" },
      { "zh": "谢谢", "py": "xiè xie", "es": "Gracias" },
      { "zh": "再见", "py": "zài jiàn", "es": "Adiós" },
      { "zh": "对不起", "py": "duì bu qǐ", "es": "Lo siento" }
    ],
    "explanation": "你好 es el saludo más común en mandarín."
  }
]
Nota: El campo audioFile debe coincidir con el nombre del MP3 que subiste. Si no tenés MP3, podés dejar el campo vacío y se usará síntesis de voz (Web Speech API).
